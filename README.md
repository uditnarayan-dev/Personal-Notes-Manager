# Notes Manager (Python CLI)

A simple command-line Notes Manager app built with Python.  
Features:
- Add, View, Search, Delete notes
- Separate categories (Work / Personal)
- Saves notes in `.txt` files
- Error handling for invalid inputs

## How to Run

1. Clone the repository:
   Run this in terminal:
   git clone git@github.com:uditnarayan-dev/Personal-Notes-Manager.git
   cd Personal-Notes-Manager
   
Make sure Python 3 is installed:
python3 --version

Run the Notes Manager:
python3 main.py

Follow the menu to add, view, search or Delete notes.
